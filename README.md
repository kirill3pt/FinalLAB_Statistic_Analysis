ОПИСАНИЕ ПРОЕКТА

Цель: Провести сквозной анализ данных о матчевой статистике игроков NBA с применением статистических методов, машинного обучения и интерпретации результатов.

Источник данных: Kaggle — NBA Player Statistics (player_data.csv).
Размер данных: 255 914 наблюдений, 115 признаков.
Тип задачи: Регрессия (предсказание количества очков PTS за матч).

ИНСТРУКЦИЯ ПО ЗАПУСКУ

Установите необходимые библиотеки:

pip install pandas numpy matplotlib seaborn scikit-learn shap
Откройте терминал в папке проекта.

Запустите Jupyter Notebook:

jupyter notebook
Перейдите в папку notebooks/ и откройте файл NBA_Players_Analysis.ipynb.
Выполните все ячейки по порядку (Cell → Run All).
================================================================================
ОСНОВНЫЕ РЕЗУЛЬТАТЫ

Модель: Random Forest Regressor (100 деревьев)

Метрики на тестовой выборке:

MAE = 1.019 очка
RMSE = 1.615 очка
R² = 0.961

Топ-3 важных признака по SHAP:

MIN — игровое время
USG_PCT — вовлечённость в атакующие действия
FG_PCT — точность бросков

Вывод: Модель хорошо прогнозирует количество очков за матч. Полученные результаты могут использоваться для аналитики игроков и оценки командных стратегий.
